# Contributing

Thanks for your interest.

This repository contains integration examples and documentation pointers for Abuse Signals API.

Rules:
- No PRs that include API keys or secrets.
- Keep examples minimal and dependency-light.
- Prefer curl + short snippets (Node / Python / C#).
- Do not add “scraping” or spam/abuse automation code.

If you want to report an issue:
- Include the request you made (without API key), the response JSON, and the requestId from meta.
- Mention whether you used explain=1 and whether the response was cached.

Support policy:
- Email is billing-only (no support).
- This repository is best-effort; no SLAs.
